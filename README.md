PicoW Smart Home Automation Controller

A robust home automation controller built around the Raspberry Pi Pico W, designed for reliable operation in real-world environments. It combines industrial communication (MODBUS over RS485) with modern IoT capabilities, making it suitable for smart homes, labs, and distributed control systems.

Features

Raspberry Pi Pico W (or compatible variants) for wireless connectivity

RS485 / MODBUS interface for industrial-grade communication

DIP switch-based MODBUS address selection for easy multi-node deployment

SHT31-DIS high-precision temperature and humidity sensor

QWIIC connector for plug-and-play I2C expansion

Reset button and user button for control and interaction

User RGB LED for status indication and debugging

2x5 SPI expansion header for additional peripherals

High-power relay (230V, 16A) for controlling appliances

Wide input voltage support: 8V to 24V DC
